# For Development
### Usage
- 開発用ビルド&画面表示
```
npm start
```

- テスト実施
```
npm run test
```

- コンポーネント表示

下記コマンドの後[http://localhost:9001](http://localhost:9001)にアクセスする。
```
npm run storybook
```

- コンポーネント用テンプレート作成
```
npm run cca
```

- 本番用ビルド
```
npm run build
```

- 開発用ビルド
```
npm run build:dev
```
